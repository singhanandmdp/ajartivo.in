Refund Policy

AJartivo provides downloadable digital design assets. Because these design files are delivered instantly and cannot be physically returned, refunds are handled under a limited, transparent policy designed for digital design purchases.

Digital Design Rules Refunds Before Download Only Customer Support for Delivery Issues

Important Notice

AJartivo follows a strict No Refund After Download policy. Please review the design preview, design details, and file format compatibility before completing payment.

Once a design file has been downloaded or accessed successfully, the purchase becomes non-refundable.

Quick Summary

1. Nature of Digital Design Files

AJartivo offers digital design files such as PSD, CDR, AI, PNG, and other downloadable assets. These are intangible design files, which means they are delivered electronically and cannot be returned like physical goods.

2. No Refund After Download

Once a user completes payment and successfully downloads the design file, no refund will be provided under any circumstances.

  • Digital files can be copied, saved, and reused after delivery.
  • Downloaded design files cannot be revoked or physically returned.
  • This policy helps protect creators, customers, and the marketplace from misuse.

3. Refund Eligibility Before Download

Refund requests may be considered only in limited situations where the purchased file was not successfully accessed or downloaded.

  • Payment was successful, but the file was not downloaded at all.
  • The design remained completely inaccessible due to a technical issue from our side.
  • A duplicate payment was made unintentionally for the same design.

For these cases, the user must contact AJartivo within 24 hours of the transaction with valid payment proof.

4. Non-Refundable Situations

  • You already downloaded the design file.
  • You changed your mind after purchase.
  • You purchased the wrong item by mistake.
  • You do not have the required software to open or edit the file.
  • You expected a different result based on personal preference.
  • There are minor visual differences between preview and final use environment, such as color or display perception.

5. Design Preview Responsibility

Before purchasing, users are expected to review the design carefully.

  • Check preview images and listing details.
  • Read the design description fully.
  • Confirm the file format matches your software and workflow.

AJartivo is not responsible for refund requests caused by incomplete review of design information before purchase.

6. Technical Issues and File Support

If you face a delivery-related problem, AJartivo will first attempt to resolve the issue through support.

  • File not downloading
  • Corrupted file
  • Wrong file received

In these situations, we may provide troubleshooting help or a replacement file. If the file has already been accessed or downloaded, the issue does not automatically qualify for a refund.

7. Fraud Prevention

Any false claim, abusive refund request, or fraudulent dispute may result in account action.

  • Permanent account suspension
  • Restriction from future purchases
  • Additional review of related transactions

8. Payment Gateway Issues

If money is deducted but no valid order is created, the issue may be caused by a payment gateway interruption.

  • The amount may be automatically reversed by the payment provider within 5 to 7 business days.
  • If the refund is not received, the user should contact their bank or payment provider.
  • AJartivo support can assist by verifying whether the order was created successfully.

9. How to Request a Refund Review

For valid cases under Section 3, please contact us with the following details:

  • Order ID
  • Payment proof or transaction reference
  • Short description of the issue
